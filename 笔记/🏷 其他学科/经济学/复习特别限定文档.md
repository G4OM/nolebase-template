# 宏经济限定文档
## 重点
1. **平衡预算乘数**：政府收入和政府购买以相等的数量增加和减少时，国民收入变动和政府收支变动的比率。
2. **凯恩斯供给曲线**：凯恩斯区域，经济严重萧条。中间区域，接近但未达到充分就业。古典区域，达到充分就业。
3. **财政政策**：变动税收和支出以影响总需求进而影响就业和国民收入的政策
4. **工资刚性**：失业率波动较大，实际工资率变化较小。**人们抵制公司下降工资**
5. **lm曲三个不同区域**：凯恩斯区域，利率极低，财政政策有效。中间区域，利率处于正常水平，财政政策和货币政策均有效。古典区域，利率非常高，货币政策有效。

1. **IS推导**：储蓄函数：  $S = -a + (1 - b)Y$。投资函数：  $I = I_0 - d r \quad$
  代入函数：

$$
I_0 - d r = -a + (1 - b)Y
$$

$$
r = \frac{I_0 + a}{d} - \frac{1 - b}{d} \cdot Y
$$
斜率：$-\frac{1-b}{d} < 0$，表明利率与收入负相关。

2. 蒙代尔弗莱明模型说明**浮动汇率制**下财政政策和货币政策对经济的影响。
财政扩张性政策，is曲线右移，利率上升，吸引国际资本，本币升值，净出口减少，is右移，政策无用。货币扩张性政策，lm右移，利率下降，资本外流，本币贬值，净出口增加，政策有用

5. **投资乘数**：k = $\frac{1}{1-b}$
6. **税收乘数**：k = $\frac{-b}{1-b}$

11. 转移支付：政府无偿支付给个人和下级政府

13. **总需求**：经济社会对产品和劳务的总需求量，等于消费需求、投资需求、政府需求、和国外需求。

18. **名义GDP**：生产的劳务和产品用**当年**的价格和产量计算的GDP结果

30. **货币市场均衡**：货币的供给与货币的需求相等

32. 利率效应：价格利率同方向变动

41. **挤出效应**：政府开支增加**引起**私人消费或投资减少
42. **滞胀**：生产停滞、失业增加、物价居高不下同时并存

## 储蓄投资恒等式
- **符号说明**：S = 储蓄，Y = 国民收入/收入/支出，C = 消费，I = 投资

- **两部门经济**（无政府、无对外贸易）  
  - 支出法：$Y = C + I$  
  - 收入法：$Y = S + C$

- **三部门经济**（考虑政府部门）G = 政府支出，T= 总税 - 政府转移支付，  
  - $Y = C + I + G$  
  - $Y = C + S + T$

- **四部门经济**  加上出口减去进口
  - $Y = C + S + T$
  - $Y = C + I + G + (X - M)$

- **总结**：事后恒等，定义恒等

## 失业&物价水平的衡量

- **劳动力人口（B）**：包括就业人口和失业人口（C）
- 不属于劳动力人口：如退休人员、学生、家务劳动者等
- **失业定义**：无工作、有能力工作、正在找工作
- **劳动年龄人口（A）**：包括劳动力人口（B）和不属于劳动力人口。
$$
\text{劳动力参与率} = \frac{\text{劳动力人数}}{\text{成年人口总数}} \times 100\% = \frac{\text{(B)}}{\text{(A)}}\times 100\%
$$
$$
\text{失业率} = \frac{\text{失业人口}}{\text{劳动力人口}} \times 100\% = \frac{\text{(C)}}{\text{(B)}}\times 100\%
$$
---

- **消费价格指数（CPI）**：反映居民购买一篮子商品和服务的价格变化  
$$
\text{CPI} = \frac{\text{现期价格指数}}{\text{基期价格指数}} \times 100
$$
- **生产者价格指数（PPI）**：衡量工业企业产品出厂价格变动趋势和程度

## 消费函数
最简单的线性**消费函数**形式为：
$$
C = a + b \cdot Y_d
$$

其中：
- $C$ —— 消费支出
- $Y_d$ —— 可支配收入（总收入减去税收）
- $a$ —— **自主消费**，即收入为零时也必须有的基本消费（>0）
- $b$ —— **边际消费倾向**，满足 $0 < b < 1$，表示每增加一单位收入所增加的消费量
-- --
- **边际消费倾向（MPC）**：$b = \frac{\Delta C}{\Delta Y_d}$
- **平均消费倾向（APC）**：$APC = \frac{C}{Y_d}$，任一收入水平上消费支出占收入的比率，随收入增加而递减

-- --
## 储蓄函数
最简单的线性储蓄函数形式为：
$$
S = Y - (a + b \cdot Y)
$$
$$
S = -a + (1 - b) \cdot Y_d
$$

其中：
- $S$ —— 储蓄
- $Y_d$ —— 可支配收入（总收入减去税收）
- $a$ —— **自主消费**（与消费函数中的 $a$ 一致）
- $b$ —— **边际消费倾向**（MPC）
- $(1 - b)$ —— **边际储蓄倾向**（MPS）$1-b = \frac{\Delta S}{\Delta Y_d}$

## IS-LM的应用
财政政策包括政府购买 $G$、税收 $T$、转移支付的变化。
- **扩张性财政政策**（增支或减税）→ 总需求增加 → IS曲线向右平移。
- **紧缩性财政政策**（减支或增税）→ IS曲线向左平移。
- **IS越平坦**（投资对利率越敏感 $d$ 大）→ 利率上升引起的投资减少多 → 挤出效应大 → 财政政策效果弱。
- **IS越陡峭**（投资对利率越不敏感）→ 挤出效应小 → 财政政策效果强。
- **LM越平坦**（货币需求的利率弹性越大，接近凯恩斯区域）→ 收入增加幅度大，利率上升幅度小 → 财政政策效果强。
- **LM越陡峭**（货币需求的利率弹性越小，接近古典区域）→ 收入增加幅度小，利率上升幅度大 → 财政政策效果弱。
---
货币政策包括中央银行调整货币供给 $M$ 或利率。
- **扩张性货币政策**（增加货币供给）→ LM曲线向右下方平移。
- **紧缩性货币政策**（减少货币供给）→ LM曲线向左上方平移。
- **IS越平坦**（投资对利率越敏感）→ 利率下降带来的投资增加越多 → 收入增加幅度大 → 货币政策效果强。
- **IS越陡峭**（投资对利率越不敏感）→ 利率下降对投资刺激小 → 收入增加幅度小 → 货币政策效果弱。
- **LM越平坦**（接近凯恩斯区域）→ 货币供给增加几乎不改变利率 → 收入增加极小 → 货币政策效果弱。
- **LM越陡峭**（接近古典区域）→ 货币供给增加引起利率大幅下降 → 投资大幅增加 → 收入增加大 → 货币政策效果强。

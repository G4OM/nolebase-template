# 📒 GTTM

这是43的博客。我在这里不定期更新一本（内容……很水的）修仙小说——《gttm》(浮慈天地生)。
-- --
- 书名：GTTM
- 吐槽：什么雷霆名字。
- 题材：穿越、日常、修仙
-- -- 
### gttm？
- 粗糙浅薄但感情真挚的中篇小说。
### 出于什么原因写它呢？
- 起因是作者的好友8敦促说：“得早点写些东西出来呀，写出个垃圾也行啊，麻利的，证明自己没有无所事事！
- ”~~作者又怎么会拒绝鞭打呢？~~
- 作者43当然是一个进取向上，昂扬积极的人。我愿意写，证明自己不是懒狗一条。（岿然且正经）
于是乎有了这篇文章，啊，你要问《门穆》去哪了？（批注作者另外一本书（坑）。）有两本看，有两个蛋糕不好吗？<br>
### 言归正传，gttm？这个是什么？
- gttm是一个架空世界的玄幻小说。
- 有修炼，有道法，有气……
- 有很多原创（雷人）设定比如**两大修仙体系：（已经over的）根时代（古法）修仙和浮慈修仙。**

总言之，故事发生在一个方兴未艾弱肉强食的浮慈世界。<br>
故事主要围绕我们的穿越佬文某（文奏迁）是如何创立脊坪山鄯水宗，是如何教育弟子（带小孩），是如何将自创的鹅情道发扬光大的故事。<br>
背景：根修士主导的根时代覆灭后，浮慈修士掌握绝对的力量和权威。黎民在门派间的夹缝下生存…… 中世纪式的思想闭塞，诸侯征伐式的乱世动荡，资本式的掠夺成性。<br>
Buff叠满了。<br>
这还是东方吗？给我干哪来了？作者傻笑，所以说是写篇架空垃圾啊，不要人参公鸡啊。（强调）
故事的主角文某人会怎么做呢？独善其身？学猴子直捣黄龙，打破天庭？这……这恐怕不行毕竟都没有中央集权（苦笑），那么学始皇帝四海一？或者闪击……？<br>

主角压力山大，何况所谓“天道”对文某人还有诸多的限制，更是捉襟见肘啊。他究竟会怎么做呢？且待下文。

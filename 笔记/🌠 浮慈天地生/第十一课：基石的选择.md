# 第十一课：基石的选择 | Zermelo-Fraenkel集合论

## 引言：为什么需要公理化集合论？

在朴素集合论基于直观的“集合”概念进行推理，即“一些确定、可区分的对象的整体”。然而，这种直观导致了悖论，如罗素悖论。为了消除悖论，同时保留集合论作为数学基础的能力，则需要一个严格的形式系统。Zermelo-Fraenkel集合论（ZFC）就是这样一套公理系统，它通过限制集合的构造方式来避免悖论，同时为数学提供足够丰富的本体。

## 一、为什么选择集合论作为基础？
一般来说,数学是一门建立在逻辑学上的学科 其与自然科学不同在于数学最多允许使用演绎推理，而自然科学允许使用演绎与归纳推理，而演绎推理是通过一个正确的前提推出一个正确的结果。可按照演绎推理,正确的前提也是由其他正确前提演绎推理得来，于是就会没有尽头,没完没了。而我们希望这条演绎链条有个起点,而不是没有起点，那么我们就设定某些前提,让这些前提不需要其他前提作支撑，也就是说,我们默认这组前提是正确的，这组前提被称为公理。数学作为一门演绎推理的学科,必然也需要一组公理。以后,我们全程采取ZFC公理体系 在这套公理下,数学的对象只有集合,集合与集合之间只有一种关系。一个集合与另一集合的唯一一个关系,就是∈(属于)。如果一个集合属于另一个集合,那么其就是另一个的元素。例如集合 A属于集合B,则记为AEB,此时A是B的元素。我们暂且把属于这一关系视为一个纸面上的主次关系。更精确的定义,由以下10条公理给出[[参考文献](https://www.bilibili.com/video/BV1xdgMzVEqb/?spm_id_from=333.337.search-card.all.click&vd_source=1b0e9e5a864b3b147c48c05dd146a88d)]

## 二、ZFC公理系统详解

-- -- 

### 1. 外延公理 (Axiom of Extensionality)
$$\forall A \forall B [\forall x (x \in A \Leftrightarrow x \in B) \Rightarrow A = B]$$
两个集合相等，当且仅当它们含有完全相同的元素。**集合完全由其元素决定**，除此之外没有其他内在属性（如“标签”或“顺序”）。

## 三、哲学辩护：为什么是ZFC？

我们并不宣称ZFC是“真理”或“唯一可能的基础”。但作为数学的基础“操作系统”，ZFC在以下三个方面达到了最佳权衡：

1. **简单性（Simplicity）**：ZFC的公理数量有限（除公理模式外，核心公理只有几条），且每条公理陈述简单直观。它们大多是对朴素集合论中合理操作的提炼。例如，配对、并集、幂集公理直接对应我们构造集合的日常方式。

2. **丰富性（Richness）**：ZFC足够强大，可以推导出几乎所有经典数学（除了少数刻意避开选择公理的构造性数学）。从自然数、实数、函数空间到代数结构、拓扑空间、泛函分析中的复杂对象，都可以在ZFC中定义并研究其性质。事实上，现代数学的绝大部分定理都可以在ZFC中形式化并证明。

3. **安全性（Safety）**：ZFC通过限制集合的构造方式（特别是分离公理模式取代无限制概括，以及基础公理禁止自属）巧妙地避免了已知的集合论悖论，如罗素悖论、康托尔悖论等。自ZFC提出以来，尚未发现内部矛盾（虽然根据哥德尔第二不完备定理，我们无法在ZFC内证明其一致性，但实践中的长期使用给了我们信心）。

## 四、ZFC的局限性

尽管ZFC非常成功，但我们仍需清醒认识其局限性：

- **不完全性**：根据哥德尔第一不完备定理，如果ZFC是一致的，则存在既不能证明也不能否定的命题（如连续统假设CH）。这意味着ZFC无法决定数学中的所有问题，我们需要在必要时接受或拒绝额外的公理（如大基数公理、力迫公理等）。

- **非唯一性**：ZFC不是数学基础的唯一选择。还有其他基础系统，如类型论、范畴论的基础、构造性集合论等。布尔巴基学派选择ZFC，部分是由于历史原因和其表达的直接性。

- **与直觉的距离**：ZFC将数学对象编码为集合，有时这种编码显得人为和笨重。例如，数字1被定义为某个特定的集合（如{{∅}}），这并不符合我们对“1”的直观。但我们应记住，编码只是实现手段，我们真正关心的是结构性质。

## 五、教学示范：从ZFC出发定义有序对

为了让学生感受ZFC的运作，我们可以展示一个经典例子：如何用集合定义有序对。

**定义（Kuratowski有序对）**：对于任意集合a,b，定义有序对为：  
`(a, b) := {{a}, {a, b}}`

**定理**：上述定义满足有序对的基本性质：`(a, b) = (c, d)`当且仅当`a = c`且`b = d`。

**证明**：
- 如果a = c且b = d，显然相等。
- 反之，假设`(a, b) = (c, d)`，即`{{a}, {a, b}} = {{c}, {c, d}}`。考虑两种情况：
  1. 如果a = b，则左边为`{{a}, {a}} = {{a}}`。那么右边也必须是单元素集，故`{c} = {c, d}`，从而c = d。且由`{{a}} = {{c}}`得`{a} = {c}`，故a = c。因此a = b = c = d。
  2. 如果a ≠ b，则左边有两个不同的元素（因为{a} ≠ {a, b}）。右边也必须有两个不同的元素，故{c} ≠ {c, d}，这意味着c ≠ d。由于左边集合等于右边集合，它们必须有相同的元素。可能出现两种情况：
     - `{a} = {c}`且`{a, b} = {c, d}`：由前者得a = c；代入后者得b = d。
     - `{a} = {c, d}`且`{a, b} = {c}`：由后者得a = b = c，但a ≠ b矛盾，故此情况不可能。
因此，总有a = c且b = d。

这样，我们仅使用集合论语言，就定义了有序对这一基本概念，并证明了其决定性性质。

## 结语

通过本课，我们期望学生理解：ZFC公理系统为数学提供了一个坚实、自洽且丰富的基础。作为布尔巴基主义者，我们承认这个基础是人为选择的，但它是一个极佳的选择，使得我们能够在其上严格地、系统地重建整个数学。在后续课程中，我们将实际演示如何从这些公理出发，一步步构造出自然数、实数、函数等对象，并展示如何将数学的各个分支纳入这一框架。

# 朴素集合论 (Naive Set Theory)

## 一、核心概念与基本运算

### 1. 集合的基本概念
*   **集合与元素**：属于关系 $a \in A$
*   **描述法**：$x \mid P(x)$ （满足性质 $P$ 的所有 $x$ 的集合）
*   **列举法**：${a_1, a_2, \dots, a_n}$
*   **子集**：$A \subseteq B \iff \forall x (x \in A \Rightarrow x \in B)$
*   **真子集**：$A \subset B \iff (A \subseteq B) \land (A \neq B)$
*   **空集**：$\emptyset =${ }，满足 $\forall x (x \notin \emptyset)$
*   **外延公理**：$A = B \iff \forall x (x \in A \Leftrightarrow x \in B)$

### 2. 集合的基本运算
*   **并集**：$A \cup B = x \mid x \in A \lor x \in B$
*   **交集**：$A \cap B = x \mid x \in A \land x \in B$
*   **差集**：$A \setminus B = x \mid x \in A \land x \notin B$
*   **补集**（相对全集 $U$）：$A^c = U \setminus A = x \in U \mid x \notin A$
*   **幂集**：$\mathcal{P}(A) = X \mid X \subseteq A$

### 3. 集合的代数定律
*   **交换律**：$A \cup B = B \cup A$，$A \cap B = B \cap A$
*   **结合律**：$(A \cup B) \cup C = A \cup (B \cup C)$，$(A \cap B) \cap C = A \cap (B \cap C)$
*   **分配律**：
    $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
    $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$
*   **德摩根定律**：
    $(A \cup B)^c = A^c \cap B^c$，
    $(A \cap B)^c = A^c \cup B^c$
*   **幂等律**：$A \cup A = A$，$A \cap A = A$
*   **吸收律**：$A \cup (A \cap B) = A$，$A \cap (A \cup B) = A$

## 二、关系与函数

### 1. 有序对与笛卡尔积
*   **有序对**（Kuratowski定义）：`(a,b) = {{a},{a,b}}`
*   **笛卡尔积**：$A \times B = {(a,b) \mid a \in A, b \in B}$
*   **$n$元组与$n$重笛卡尔积**：$A_1 \times A_2 \times \dots \times A_n = {(a_1, a_2, \dots, a_n) \mid a_i \in A_i}$

### 2. 关系
*   **二元关系**：$R \subseteq A \times B$
*   **定义域与值域**：
    $\text{dom}(R) = {a \in A \mid \exists b \in B: (a,b) \in R}$，
    $\text{ran}(R) = {b \in B \mid \exists a \in A: (a,b) \in R}$
*   **逆关系**：$R^{-1} = {(b, a) \mid (a, b) \in R}$
*   **关系的复合**：若 $R \subseteq A \times B$，$S \subseteq B \times C$，则 $S \circ R = {(a,c) \mid \exists b \in B: (a,b) \in R \land (b,c) \in S}$

#### 特殊关系类型：
*   **等价关系**（$\sim$ 满足）：
    1.  **自反性**：$\forall a \in A,\ a \sim a$
    2.  **对称性**：$\forall a, b \in A,\ a \sim b \Rightarrow b \sim a$
    3.  **传递性**：$\forall a, b, c \in A,\ (a \sim b \land b \sim c) \Rightarrow a \sim c$
*   **等价类**：$[a] = {x \in A \mid x \sim a}$
*   **商集**：$A/{\sim} = {[a] \mid a \in A}$
*   **序关系**：
    *   **偏序** $\leq$：自反、反对称（$a \leq b \land b \leq a \Rightarrow a = b$）、传递
    *   **全序**：偏序且任意两元素可比（$\forall a, b,\ a \leq b \lor b \leq a$）
    *   **良序**：全序且每个非空子集有最小元

### 3. 函数
*   **函数定义**：$f: A \to B$ 是满足以下条件的二元关系：
    1.  $\text{dom}(f) = A$
    2.  $\forall a \in A, \forall b_1, b_2 \in B: ((a, b_1) \in f \land (a, b_2) \in f) \Rightarrow b_1 = b_2$
    （记作 $f(a) = b$ 当 $(a, b) \in f$）
*   **函数类型**：
    *   **单射**：$\forall a_1, a_2 \in A,\ f(a_1) = f(a_2) \Rightarrow a_1 = a_2$
    *   **满射**：$\forall b \in B, \exists a \in A: f(a) = b$
    *   **双射**：既是单射又是满射
*   **复合函数**：若 $f: A \to B,\ g: B \to C$，则 $(g \circ f): A \to C,\ (g \circ f)(a) = g(f(a))$
*   **反函数**：若 $f$ 是双射，则存在 $f^{-1}: B \to A$ 满足 $f^{-1}(b) = a \iff f(a) = b$，且 $f^{-1} \circ f = \text{id}_A$，$f \circ f^{-1} = \text{id}_B$

## 三、无穷与基数（康托尔理论）

### 1. 等势
*   **定义**：集合 $A$ 与 $B$ 等势（记作 $A \approx B$），如果存在双射 $f: A \to B$
*   **例子**：
    *   $\mathbb{N} \approx \mathbb{Z} \approx \mathbb{Q}$（可数集）
    *   $(0, 1) \approx \mathbb{R}$（连续统）

### 2. 有限集与无穷集
*   **戴德金无穷定义**：集合 $A$ 是无穷的当且仅当存在真子集 $B \subset A$ 使得 $A \approx B$
*   自然数集 $\mathbb{N}$ 是“最小”的无穷集

### 3. 基数
*   **基数**：集合“大小”的度量，等势的集合有相同的基数
    *   有限集基数：$|{a_1, \dots, a_n}| = n$
    *   $|\mathbb{N}| = \aleph_0$（阿列夫零）
    *   $|\mathbb{R}| = \mathfrak{c}$（连续统基数）
*   **基数比较**：$|A| \leq |B|$ 当存在单射 $f: A \to B$
*   **康托尔-伯恩斯坦定理**：$(|A| \leq |B| \land |B| \leq |A|) \Rightarrow |A| = |B|$
*   **康托尔定理**：对任意集合 $A$，$|A| < |\mathcal{P}(A)|$
    *   特别地，$\aleph_0 = |\mathbb{N}| < |\mathcal{P}(\mathbb{N})| = 2^{\aleph_0} = \mathfrak{c}$

### 4. 基数算术
对于基数 $\kappa, \lambda$：
*   **加法**：若 $A \cap B = \emptyset$，则 $|A \cup B| = \kappa + \lambda$
*   **乘法**：$|A \times B| = \kappa \cdot \lambda$
*   **指数**：$|B^A| = \lambda^\kappa$（其中 $B^A = {f: A \to B}$）

**重要结果**：
$\aleph_0 + \aleph_0 = \aleph_0,\quad \aleph_0 \cdot \aleph_0 = \aleph_0,\quad 2^{\aleph_0} = \mathfrak{c}$

## 四、序数与良序

### 1. 良序集
*   **定义**：全序集 $(A, \leq)$ 中，每个非空子集都有最小元
*   **例子**：$(\mathbb{N}, \leq)$ 是良序集，$(\mathbb{R}, \leq)$ 不是良序集

### 2. 序型
*   **序同构**：两个全序集 $(A, \leq_A)$ 和 $(B, \leq_B)$ 同构，如果存在双射 $f: A \to B$ 保持序结构：
    $\forall x, y \in A: x \leq_A y \iff f(x) \leq_B f(y)$
*   **序型**：同构良序集的等价类

### 3. 序数（朴素理解）
*   **冯·诺依曼序数**（直观描述）：
    $0 = \emptyset,\ 1 = {0} = {\emptyset},\ 2 = {0,1} = {\emptyset, {\emptyset}},\ \dots$
    $\omega = {0,1,2,\dots} = \mathbb{N},\ \omega+1 = \omega \cup {\omega},\ \dots$
*   每个序数都是所有小于它的序数的集合

## 五、悖论

### 1. 罗素悖论（1901）
*   **构造**：令 $R = {x \mid x \notin x}$
*   **推理**：$R \in R \iff R \notin R$，矛盾

### 2. 康托尔悖论（1899）
*   **构造**：令 $V$ 为所有集合的集合
*   **推理**：由康托尔定理，$|\mathcal{P}(V)| > |V|$，但 $\mathcal{P}(V) \subseteq V$，矛盾

### 3. 布拉利-福尔蒂悖论（1897）
*   **构造**：令 $\text{Ord}$ 为所有序数的集合
*   **推理**：$\text{Ord}$ 是良序集，应有序数 $\Omega$，但 $\Omega \in \text{Ord}$ 且 $\Omega < \Omega$，矛盾

## 朴素集合论 vs 公理化集合论

| 特征 | 朴素集合论 | 公理化集合论（如ZFC） |
| :--- | :--- | :--- |
| **基础** | 基于直观的“集合”概念 | 基于明确的公理系统 |
| **概括原则** | 无限制：对任意性质 $P$，${x \mid P(x)}$ 是集合 | 受限：分离公理模式 ${x \in A \mid P(x)}$ |
| **悖论处理** | 包含已知悖论 | 通过公理设计避免已知悖论 |
| **应用** | 19世纪末-20世纪初数学的基础语言 | 现代数学的标准基础 |

### 历史意义
朴素集合论虽然因悖论而被公理化集合论取代，但其历史贡献不可忽视：
*   为分析学提供了严格基础（实数理论、函数空间）
*   发展了无穷集合的数学理论（基数、序数）
*   统一了数学语言，促进了20世纪数学的结构化发展

> **注意**：现代数学教育中学习的“集合论”通常是在公理化框架（ZFC）下讲述的，但内容上仍包含朴素集合论的大部分核心概念（只是构造上更加谨慎）。

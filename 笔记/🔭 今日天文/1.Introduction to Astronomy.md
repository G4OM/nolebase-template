### 1. 人类的认知起源

- **核心命题**：文明诞生之初，人类对夜空的凝视催生最古老的科学——天文学
    
- **历史脉络**：
    
    - 可追溯至美索不达米亚文明时期（约公元前3500年）
        
    - 早期天文学与神秘主义交织，通过神话叙事解释宇宙现象
        
    - 古代天象记录成为最早的实证数据采集活动

### 2. 天文学的发展阶段

| 时期   | 主要特征        |
| ---- | ----------- |
| 古代   | 绘制星图，天体运动预测 |
| 科学革命 | 地心说转向日心说    |
| 现代   | 天体物理学和宇宙学诞生 |
### 3. 二十一世纪对宇宙的认知

- **已解谜题**：
    
    - 宇宙年龄测定（约138亿年）
        
    - 大爆炸理论（Big Bang Theory）的观测证据
        
        - 宇宙微波背景辐射（CMB）
            
        - 哈勃定律（Hubble’s Law）
            
- **待解问题**：
    
    - 暗物质（Dark Matter）与暗能量（Dark Energy）的本质
        
    - 宇宙终极命运（大撕裂/热寂说）
        

### 4. 系列课程结构

1. **基础模块**（形成时序）：
    
    - 能量与基本粒子
        
    - 恒星与星系的演化
        
    - 行星系统形成
        

2.**进阶专题**：

- 黑洞与引力波
    
- 系外行星探测
    
- 多重宇宙理论
